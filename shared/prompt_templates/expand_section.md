# 論文章節擴寫協定（Expand Section Protocol）

> 適用場景：將 overall_review 中的某一節摘要，擴寫為 1,500-2,000 字的學術段落。

## 輸入（每次執行前必須讀取）
1. `00_overall_review/` → 找到本章節的定位段落與核心論點
2. 目標章節的 `notes.md` → 找到擴寫優先順序
3. 上一章的 `draft.md` 結尾段 → 取得邏輯銜接點
4. `shared/glossary.md` → 確認術語使用一致性
5. `shared/references_master.bib` → 引用必須來自本庫

## 輸出規範
- 字數：1,500-2,000 繁體中文
- 段落結構：
  - 開門句：本節要回答什麼問題（1 句）
  - 主體：3-4 個論證段落，每段有主題句 + 證據/文獻支持
  - 銜接句：與下一節的邏輯橋接
- 引用：至少嵌入 2-3 個文獻（APA 格式），必須來自 `references_master.bib`
- 結尾句：須明確包含「本節論點與 overall_review 的對應關係」
- 語調：學術嚴謹。禁止行銷口吻、禁止第一人稱主觀判斷（「我認為」「我們覺得」）

## 禁止事項
- 不得引入 `overall_review` 中未定義的理論變數
- 不得憑空創造文獻引用
- 不得改變已確立的術語定義
- 不得偏離本章在 `overall_review` 中的定位

## 輸出後檢查
- [ ] 字數是否在 1,500-2,000 範圍內？
- [ ] 是否至少引用 2-3 個文獻？
- [ ] 結尾句是否回應 overall_review？
- [ ] 是否使用 glossary 中的標準術語？
