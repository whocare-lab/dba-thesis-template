# AMJ 論文寫作框架 — CH1-CH3 核心摘要

> 來源：Academy of Management Journal 編輯群文章發表系列
> 用途：DBA 課程報告、研究計畫撰寫時套用 AMJ 審查標準自我檢核

## Ch1：主題選擇（Topic Selection）— 五大標準

| 標準 | 核心問題 |
|:---|:---|
| **重要性** | 是否挑戰重大挑戰（grand challenge）？ |
| **新穎性** | 是否改變對話方向？ |
| **奇特性** | 是否激發好奇心？ |
| **範疇廣度** | 視角是否夠宏大？ |
| **實用性** | 能否提供實務洞見？ |

## Ch2：研究設計（Research Design）— 三大致命傷

1. **研究問題與設計不匹配**：問「如何」（how）→ 用質性個案，不是量化大樣本
2. **測量與操作化問題**：必須清楚區分相似但不同的構念
3. **模型設定不完善**：必須納入替代解釋並逐一排除

**最佳實踐**：多研究、多樣本設計（如結合實地研究與實驗室實驗）

## Ch3：前言撰寫（Writing the Introduction）— 三問

1. **誰在乎？（Why care?）**：學術社群、實務界、政策制定者
2. **已知、未知，知道了又如何？**：用「不充分法」定位——不貶低前人，也不過激攻擊
3. **我們可以學到什麼？**：預告 2-3 個理論命題

**常見失誤**：空泛的「填補研究缺口」當理由、前言冗長無聚焦、過度承諾無法兌現

## 與 CIMO 框架的結合

AMJ 第二章的研究設計可以直接對應到 CIMO 四格：

- **C（情境）** = 研究場域的選擇理由
- **I（干預）** = 個案中的策略行動
- **M（機制）** = 理論要解釋的因果鏈
- **O（結果）** = 可觀察的策略成果
