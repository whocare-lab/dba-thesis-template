# EMBA學習支持生態系統之建構

> 專案定位：羅懷均教授子計畫「EMBA學習支持生態系統之建構：資源治理、組織能力累積與產學共創」

## 核心宗旨

探討如何打造「對學生好、對學生有幫助」的 EMBA 學習支持生態系統。

## 核心對象

- 在校生
- 校友
- 師資

## 支持面向

### 1. 學生支持（硬）
- 實體場地
- 國內外通訊與討論空間
- 學習活動與操作資源

### 2. 學生支持（軟）
- 數位平台設計
- 輔導學生學習
- 依不同學習者提供不同學習清單
- 量化回饋與學習成效評估
- 論文教戰手冊與學習診斷

### 3. 行動式診斷
- 扮演 EMBA 學習設計者
- 強調診斷與輔導
- 不是企業參訪

### 4. 師資支持
- 教學配套
- 師資輔導
- 教學支持機制

## 執行方法

1. 深度訪談（質性）
2. 問卷調查（量化）
3. 標竿研究（新加坡等海外案例）
4. 學術與政策產出（TSCI 特刊、研討會、政策白皮書）

## 後續整理方向

- 子計畫問題意識
- 理論架構與變數對應
- CIMO 分析
- 個案 / 問卷 / 標竿研究設計
- 政策與學術產出路徑
