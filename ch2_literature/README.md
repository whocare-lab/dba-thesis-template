# 第二章：文獻探討

> 目的：建立研究的「理論基礎」— 誰說過什麼？他們漏了什麼？你的研究補什麼？

## 三階段流程

### 階段 A：文獻檢索
1. 定義核心理論關鍵字
2. 使用 Crossref / Semantic Scholar API 搜尋
3. 過濾條件：被引 > 50、主流期刊、近 5 年
4. 產出：文獻檢索清單

### 階段 B：文獻蒸餾
1. 每篇文獻生成一張 CIMO 蒸餾卡
2. 卡中包含：Context / Intervention / Mechanism / Outcome
3. 你只需寫「與本研究的連結」（50-100 字/卡），其餘 AI 包辦
4. 產出：15-25 張蒸餾卡 → `shared/distilled_cards/`

### 階段 C：縱橫分析
1. 橫向比較：同理論內文獻的一致性/矛盾
2. 縱向演進：理論從奠基到最新的演變
3. 跨理論對話：理論間的互補或衝突
4. 缺口矩陣：哪些 CIMO 組合尚未被研究？
5. 產出：缺口分析報告


## 經典文獻豁免通道
以下文獻類型不受期刊排名限制：
- 被引 > 1000 的奠基文獻 → 自動豁免
- 理論原創書籍章節 → 人工審核
- 產業實務報告 → 標記 [Industry Source]
- 中文經典文獻 → 標記 [Chinese Classic]
