# 第一章：緒論

> 目的：建立研究的「正當性」— 為什麼這個問題值得研究？為什麼現在？為什麼是你？

## 所需檔案

| 檔案 | 用途 | 狀態 |
|------|------|------|
| `draft.md` | 初稿（自行撰寫或 AI 輔助生成） | 待建立 |
| `notes.md` | 討論筆記、教授反饋、待辦事項 | ✅ 已建立 |

## 撰寫指引

對照 AMJ 選題五大標準自我檢核：
1. **重要性**：挑戰了什麼重大議題？
2. **新穎性**：改變了什麼既有對話方向？
3. **奇特性**：為什麼讓人好奇？
4. **範疇廣度**：視角是否夠宏大？
5. **實用性**：對實務界有什麼洞見？

## 建議結構
```
1.1 研究背景與動機
1.2 研究問題（主問題 + 2-3 子問題）
1.3 研究目的與預期貢獻
1.4 研究範圍與限制
1.5 論文結構
```
