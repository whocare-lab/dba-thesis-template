# 第三章：研究方法

> 目的：說服讀者你的研究設計足夠嚴謹，足以回答 CH1 的研究問題。

## 方法論基礎
- Yin (2018)：個案研究設計
- Eisenhardt (1989)：從個案建構理論
- Langley (1999)：過程資料分析策略
- Gioia et al. (2013)：質性研究嚴謹度

## 訪談文件五件組

按執行順序排列，皆為通用模板，填入受訪者資訊即可使用：

| # | 檔案 | 用途 | 使用時機 |

| 1 | `invitation_letter.md` | 正式訪談邀請 | 約訪前 1-2 週 |

## 建議結構
```
3.1 研究哲學與方法論立場
3.2 個案研究法
    3.2.1 個案選擇邏輯
    3.2.2 個案背景描述
3.3 資料收集方法
    3.3.1 初級資料（訪談）
    3.3.2 次級資料（文件、年報）
3.4 資料分析方法
    3.4.1 CIMO 框架應用
3.5 研究品質檢核（構念/內部/外部效度 + 信度）
3.6 研究倫理
```

## 品質檢核表（四項標準）

| 檢核標準 | 策略 | 執行方式 |
|---------|------|---------|
| 構念效度 | 多元證據來源 | 訪談 + 年報 + 法說會 + 新聞 |
| 內部效度 | 模式匹配 | CIMO 框架編碼 → 命題歸納 |
| 外部效度 | 理論抽樣 | 選擇典型/極端/關鍵個案 |
| 信度 | 個案協議書 + 審蹤 | 使用 `case_protocol.md` + `review_changelog.md` |
