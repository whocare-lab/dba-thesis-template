## 3.1 研究哲學與方法論立場

### 3.1.1 認識論立場
- 你採用什麼認識論？（詮釋主義／實證主義／實用主義）
- 為什麼？
- 對你的研究設計有何意涵？

### 3.1.2 本體論立場
- 你採用什麼本體論？（批判實在論／建構主義／客觀實在論）
- 為什麼？
- 對你的研究設計有何意涵？

### 3.1.3 方法論選擇：為何選擇質性研究？

1. **探索性**：你的研究問題是否具有探索性質？
2. **過程性**：你的研究關注「過程」而非「狀態」？
3. **情境性**：你的研究是否需要深度脈絡化理解？
4. **複雜性**：你的核心概念是否難以量化測量？

> 引用 Yin (2018) 或 Eisenhardt (1989) 為你的選擇辯護。

## 3.4 資料分析方法

### 3.4.1 分析框架：演繹—歸納混合編碼

1. **第一階段：演繹編碼**
   - 基於文獻建立先驗編碼架構

| 編碼維度 | 對應理論 | 子編碼示例 |
|---------|---------|-----------|
| [維度一] | [理論 A] | [範例] |
| [維度二] | [理論 B] | [範例] |
| ... | ... | ... |

2. **第二階段：歸納編碼**
   - 捕捉先驗架構未涵蓋的新興主題

3. **第三階段：軸心編碼**
   - 建立範疇之間的關係

4. **第四階段：選擇性編碼**
   - 建立核心範疇，整合為理論故事線

### 3.4.2 分析工具

| 工具 | 用途 |
|------|------|
| NVivo / Atlas.ti | 質性資料管理（或 Excel 替代） |
| 時間軸矩陣 | 事件時間排列，對照理論 |
| 因果網絡圖 | 事件—機制—結果因果鏈 |
| 概念圖 | 核心概念關係網絡 |

### 3.4.3 分析流程

```
Step 1: 資料準備（逐字稿、文件數位化、建立資料庫）
Step 2: 開放編碼（逐行閱讀、賦予初始編碼）
Step 3: 軸心編碼（歸類為範疇、建立關係）
Step 4: 選擇性編碼（確認核心範疇、整合故事線）
Step 5: 理論驗證（與文獻對話、Member checking）
```

## 3.5 研究品質標準

| 量化標準 | 質性對應 | 本研究的做法 |
|---------|---------|------------|
| 內在效度 | 可信性 | 三角驗證、長期投入、同儕審視 |
| 外在效度 | 可轉移性 | 豐富脈絡描述（thick description） |
| 信度 | 可確認性 | 審查軌跡（audit trail） |
| 客觀性 | 可信賴性 | 研究者反思（reflexivity） |
