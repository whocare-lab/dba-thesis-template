## 二、受訪者權利（請逐項確認）

作為本研究之受訪者，您擁有以下權利：

| | 權利項目 | 說明 |
|---|---------|------|
| ☐ | **自願參與** | 參與本研究完全出於自願，無任何強制或利益交換 |
| ☐ | **隨時中止** | 訪談過程中，您有權隨時中止，不需提供任何理由 |
| ☐ | **拒絕回答** | 任何問題若您不願回答，可直接跳過，不需解釋 |
| ☐ | **內容刪除** | 訪談結束後 30 天內，您有權要求刪除特定發言或整份訪談資料 |
| ☐ | **匿名處理** | 您的姓名、職稱、及其他可識別資訊將於論文中完全匿名化（以代號呈現） |
| ☐ | **逐字稿確認** | 訪談錄音轉錄為逐字稿後，將提供您確認與修正，確認後方納入分析 |
| ☐ | **發表前審閱** | 論文中直接引用您發言的段落，將於投稿前提供您審閱 |
| ☐ | **資料撤回** | 若您事後決定退出研究，已蒐集的資料將全數銷毀，不納入分析 |

本人（研究者）承諾：

1. **錄音資料**：加密儲存於個人電腦，不存放於公有雲端空間。研究完成後三年內銷毀。
2. **逐字稿**：去識別化處理後方進行分析。原始逐字稿僅研究者本人可存取。
3. **使用範圍**：僅用於本博士論文及相關學術發表。絕不用於商業顧問或營利目的。
4. **資料共享**：匿名化後的分析資料，可能依學術慣例在發表時提供給期刊審查人。
