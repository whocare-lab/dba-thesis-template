# 逐字稿確認信模板

> **使用方式**：訪談後一週內，完成逐字稿後寄出。填入 `[ ]` 欄位即可。
> **法律意義**：口試委員質疑「受訪者知情嗎」時的書面證據。

**附件說明**

- 總字數：約 [X,XXX] 字
- 預計引用：約 [XX] 段（已在逐字稿中以底線標註）

**確認請求**
請您撥冗閱讀，並確認以下事項：
1. 引述內容是否準確反映您的原意？
2. 您的身份代號「[代號]」是否可接受？（若需調整請告知）
3. 是否有任何段落需要刪除、修改或補充？

**如何標註修改**
請直接在 PDF 上標註，或使用以下格式回覆：

| 動作 | 標註方式 | 範例 |
|------|---------|------|
| 刪除 | `【刪】` | 「那段文字【刪】」 |
| 修改 | `【改：修正後文字】` | 「那段文字【改：新版本】」 |
| 補充 | `【補：補充內容】` | 「【補：當時還有另一個考量...】」 |
| 代號調整 | `【代號：新建議】` | 「【代號：技術副總】」 |

## 實務備註（給研究者）

### 附整份逐字稿還是只附引述摘要？

| 方案 | 優點 | 缺點 | 適用對象 |
|------|------|------|---------|
| 整份逐字稿 | 完全透明 | 字多，可能沒耐心看完 | 關係近、時間充裕 |
| 引述摘要表 | 高效 | 看不到完整脈絡 | 高階主管、時間有限 |

**建議**：寄整份逐字稿，但信中說明「預計引用的段落已用底線標註，若時間有限可優先確認」。

### 受訪者要求刪除關鍵段落？
- 尊重決定（協議書賦予的權利）
- 可禮貌詢問：「是否可以用更概括的方式呈現？」
- 若堅持刪除 → 找其他資料來源補強同一論點

### 保存期限
- 逐字稿、確認信、回覆：至少保存到口試後三年
- 錄音檔：口試後可銷毀，建議保留到畢業後一年

```

訪談後 24h：thankyou_letter_template.md  → 感謝 + 預告逐字稿
訪談後 1 週：本文件                      → 寄逐字稿供確認 ✉️
訪談後 2-3 週：受訪者回覆修正           → 更新逐字稿

```
